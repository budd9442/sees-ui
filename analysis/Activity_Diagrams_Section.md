# 3.5 Activity Diagrams for Proposed System

For each of the critical system processes in the Student Enrollment and Evaluation System (SEES), activity diagrams have been developed to show the flow of events for processes such as bulk enrollment, pathway selection, module registration, grade management, and performance tracking. Each of these diagrams has outlined the steps that the users and the system take in managing a specific academic task; therefore, visually showing how each function is realized within the academic workflow. These diagrams demonstrate the interaction between different actors (students, academic staff, administrators) and the system components, ensuring comprehensive coverage of all major academic processes from enrollment to graduation.
